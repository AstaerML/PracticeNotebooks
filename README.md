# Программирование Систем Исскуственного Интелекта - Практические работы.
В данном репозитории хранятся работы связанные с ML и DL, которые были выполнены по учёбе с целью эксперементального изучения и наработки навыком программирования.
Все работы были выполнены в программе Jupyter Notebook и хранятся в формате ".pynb".
